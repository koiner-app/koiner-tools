

# Koiner Tools

## Docker-compose

### Run nx builds
```
npm run build
```

### Start docker containers
```
docker-compose up
```

### Add virtual hosts (Optional)
```
127.0.0.1	checker.local
```

## Kubernetes

Run nx builds
```
npm run build
```

Start minikube
```
minikube start
```

Deploy cluster
```
kubectl apply -f ...
```
